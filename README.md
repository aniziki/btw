# btw

Small reminder/usage utilities.

## Usage

Show usage stats:

```bash
python3 ./usage.py
```

Set a reminder from a file:

```bash
python3 ./btw-reminder.py set my-reminder.txt
```

## Patched binary

Run the patched binary at:

```
/home/<user>/.local/share/claude/versions/<version>.btw-patched
```

Model: `fable`
